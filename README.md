# Diwali-Sales-Analysis-and-Reporting
**This project analyzes customer purchasing behavior during the Diwali festival using Python and its libraries.** 
**This project analyzes customer purchasing behavior during the Diwali festival using Python and Excel.**
**It explores sales patterns across gender, age groups, product categories, and states to identify top-performing segments.**
**The project includes a synthetic dataset of 500 records, Python-based analysis scripts, and a PowerPoint report highlighting key insights, visualizations, and business recommendations.**
